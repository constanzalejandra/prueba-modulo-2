HTML y CSS para principiantes

HTML y CSS son dos herramientas importantes para dar inicio en el undo del desarrollo web. Por lo tanto, en este curso aprenderás todo sobre HTML y CSS, desde lo más básico hasta el nivel intermedio, logrando desarrollar tu primera página web responsiva, sin la necesidad de aplicar librerías externas para css. Instructor: Manuel González

Estilos con sass

SASS es una herramienta importante hoy en dia para todo desarrollador web, ya que permite crear estilos mas ordenados, eficientes y con menos lineas de código. Por ello, en este curso aprenderás todo lo necesario para crear jojas de estilos desde la sintaxis de SCSS, implementando técnicas propias de SASS. Instructor: Manuel González
